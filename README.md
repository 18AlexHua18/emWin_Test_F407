# 项目介绍
本项目用于建立一个模板工程, 便于后期查看与快速适配emWin库

库版本: V5.48

分支emWin6: v6.1.0

# 发生过的问题

若进行工程迁移, 注意删除这个配置, 使用cubeMX生成的cmakelist即可
